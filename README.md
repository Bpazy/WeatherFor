WeatherFor
======
SMS or email to remind the weather at regular time. 

## Attention
Only support chinese cell phone number.

## Usage
```
{
  "nums": [
    {
      "num": "18651242070", // cell phone number
      "location": "nanjing" // cell phone number's location
    }
  ],
  "hour": 12, // Start time.
  "minute": 12,
  "second": 30,
  "interval": 24 // Interval time(/h)
}
```
